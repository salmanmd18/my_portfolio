# Salman Portfolio

## Running locally

- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`

## Deploying to GitHub Pages

This project uses `gh-pages` to publish the built `dist` folder.

- Deploy: `npm run deploy`
- URL: `https://GITHUB_USERNAME.github.io/salman-portfolio/`

## Deployment (Vercel)

- Build command: `npm run build`
- Output directory: `dist`
- Dev command: `npm run dev`
